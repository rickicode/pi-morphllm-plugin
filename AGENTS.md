# Repository Instructions

- Before making any edit in this repo or plugin, always ask the user for confirmation first.
- Do not create or update `CHANGELOG.md` immediately when a change is still tentative, exploratory, or not yet final.
- When a change is confirmed as final, always create or update `CHANGELOG.md` so the version history records what changed.
- Treat changelog updates as required for any finalized repository or plugin edit, not only version bumps.
